User Guide
Start the System.
1. Add New Asset
Select “Add Asset” from the menu.
Enter the required details:
Asset Name
Asset ID
Quantity
Department
Save the asset to store it in the system.
2. View Assets
Choose “View Assets”.
The system will display the list of all available assets with their details.
3. Update Asset Information
Select “Update Asset”.
Enter the Asset ID of the item you want to edit.
Input the new details (name, quantity, or department).
Save changes.
4. Delete Asset
Select “Delete Asset”.
Enter the Asset ID.
The system will remove the selected asset from the list.
5. Search Asset (Optional)
Choose “Search Asset” if available.
Enter the asset name or ID.
The system will show the matching result.
6. Exit the System
Select “Exit” from the menu.
The program will close safely.
